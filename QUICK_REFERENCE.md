## 🐛 Troubleshooting

**Page doesn't load?**
```bash
npm cache clean --force
rm -rf node_modules
npm install
npm start
```

**Animations choppy?**
- Update Chrome/Firefox/Safari

**Menu not opening?**
- Clear browser cache (Ctrl+Shift+Delete)

**3D DNA not showing?**
- Check internet (Three.js loads from CDN)
